## shortcuts

### buyers and sellers
```
q | a : rb +/- 5

w | s : ib +/- 5

e | d : is +/- 5

r | f : rs +/- 5

shift modifier for +/- 1
```

### speed
```
t | up arrow : speed +

t | down arrow : speed -

p : pause
```

### presets
```
0 : set all to 50

- : set all to 0

= : set all to 100

space : reset quote

esc : reset everything
```

### sharing
```
c : copy to clipboard
```
