# imbalancewalk

## shortcuts

### buyers and sellers
```
Q|A: rb +/- 5

W|S: ib +/- 5

E|D: is +/- 5

R|F: rs +/- 5

SHIFT modifier: +/- 1
```

### speed
```
T|⬆: speed +

G|⬇: speed -

P: pause
```

### presets
```
0: set all to 50

-: set all to 0

=: set all to 100

SPACE: reset quote

ESC: reset everything
```

### utility
```
H: toggle visibility
C: copy to clipboard
```

### help
```
?: readme
```
